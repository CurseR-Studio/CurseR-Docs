Audio Source Pool

AudioSource 컴포넌트를 미리 생성해두고 재사용하는 풀. AudioEmitter가 내부적으로 사용하며, 독립적으로도 활용 가능하다.

생성

AudioSourcePool은 일반 C# 클래스로, GameObject를 받아 초기화한다.

var pool = new AudioSourcePool(gameObject, initialSize: 4);
파라미터 설명
owner AudioSource 컴포넌트를 붙일 GameObject
initialSize 초기 풀 크기 (기본값 4)

빌리기 / 반납

// 풀에서 AudioSource 빌려옴 (풀이 비면 자동 확장)
AudioSource source = pool.Get();

// 사용 후 반납 (Stop + 초기화, Destroy 없음)
pool.Release(source);
Warning

Get()으로 빌린 AudioSource는 반드시 Release()로 반납해야 한다. 반납하지 않으면 풀이 고갈되어 계속 새로 생성된다.
