Audio Emitter

상호작용 오브젝트에서 효과음과 루프 사운드를 재생하는 컴포넌트. AudioSourcePool을 내부적으로 사용해 루프 사운드의 AudioSource를 재사용한다.

설정

GameObject에 AudioEmitter 컴포넌트를 추가하면 AudioSourcePool이 자동으로 함께 추가된다. Inspector의 Sounds 배열에 키-클립 쌍을 등록한다.

필드 설명
key 코드에서 사운드를 식별하는 문자열
clip 재생할 AudioClip
volume 볼륨 (0~1, 기본값 1)

Object에서 사용하기

Object를 상속하는 컴포넌트에서는 useSound()로 AudioEmitter를 캐싱해 사용한다.

private AudioEmitter _sound;

protected override void Init()
{
    _sound = useSound();
}

효과음 재생

겹쳐서 재생 가능. 재생 중 Stop 불가.

_sound?.Play("hit");

루프 사운드

// 시작
_sound?.PlayLoop("ambient");

// 특정 키 중지
_sound?.StopLoop("ambient");

// 전체 중지
_sound?.StopAllLoops();
Note

같은 키로 PlayLoop()를 중복 호출하면 무시된다.

상속 확장

메서드를 override해 동작을 확장할 수 있다.

public class ButtonAudioEmitter : AudioEmitter
{
    public override void Play(string key)
    {
        base.Play(key);
        // 추가 처리 (예: 파티클 연동)
    }
}
