Shared Flag

공유 boolean 플래그.
점유 중인 영역에서 모두 사용 시 자동으로 기본값으로 리셋됨.

선언

var defaultValue = false;
var flag = new SharedFlag(defaultValue);

점유 및 사용

비동기적으로 동시에 다중 점유 가능.
점유 중인 영역에서 모두 handle.GetValue()가 호출되면 자동으로 기본값으로 리셋됨.

using (var handle = flag.Acquire()) {
    var val = handle.GetValue();
    // Do something
}
// 자동으로 점유 해제됨

플래그 설정하기

기본값의 반댓값으로 플래그 설정

flag.Set();
